# Superpowers Work Folder

Use `plans/` and `specs/` only for active or intentionally deferred work.

Completed design specs and implementation plans should be moved to
`archive/completed/specs/` and `archive/completed/plans/` respectively. Keep the
original filenames so commit history and cross-references remain easy to trace.

Before archiving, check at least one completion signal:

- implementation commits exist for the plan/spec scope,
- current code and tests expose the planned surface, or
- follow-up docs explicitly mark the roadmap item complete.

Leave broad backlog or deferred umbrella plans in the active folder until a
replacement roadmap or explicit cancellation exists.
